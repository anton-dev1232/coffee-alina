# ☕ Сайт "Кофе с Полиной" — инструкция

## Что нужно сделать ДО запуска

### Шаг 1 — Создать Telegram-бот

1. Открой Telegram, найди **@BotFather**
2. Напиши `/newbot`
3. Придумай имя бота (например: `Полина Кофе`) и username (например: `polina_coffee_bot`)
4. BotFather пришлёт **токен** — длинная строка вида:
   ```
   7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx
   ```
5. Открой созданный бот и нажми **Start** (чтобы он мог тебе писать)

### Шаг 2 — Узнать свой Chat ID

1. Напиши любое сообщение своему боту
2. Открой в браузере ссылку (подставь свой токен):
   ```
   https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates
   ```
3. В ответе найди `"chat":{"id":XXXXXXXX}` — это и есть твой **chat_id**

### Шаг 3 — Вставить данные в `index.html`

Открой `index.html` и найди строки:
```js
const TG_TOKEN = 'ВАШ_ТОКЕН_БОТА';
const TG_CHAT  = 'ВАШ_CHAT_ID';
```
Замени на свои значения, например:
```js
const TG_TOKEN = '7123456789:AAFxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx';
const TG_CHAT  = '123456789';
```

## Запуск локально (для проверки)

Просто открой файл `index.html` в браузере двойным кликом — всё работает без сервера.

## Деплой — выложить сайт в интернет

### Вариант А: GitHub Pages (бесплатно, просто)

1. Зарегистрируйся на [github.com](https://github.com) (если нет аккаунта)

2. Создай новый репозиторий:
   - Нажми **+** → **New repository**
   - Имя: `coffee-polina` (или любое)
   - Visibility: **Public**
   - Нажми **Create repository**

3. Загрузи файл:
   - На странице репозитория нажми **uploading an existing file**
   - Перетащи `index.html`
   - Нажми **Commit changes**

4. Включи GitHub Pages:
   - Зайди в **Settings** → **Pages**
   - Source: **Deploy from a branch**
   - Branch: **main**, папка: **/ (root)**
   - Нажми **Save**

5. Через 1-2 минуты сайт будет доступен по адресу:
   ```
   https://ВАШ_ЛОГИН.github.io/coffee-polina/
   ```

---

### Вариант Б: Netlify (бесплатно, ещё проще)

1. Зайди на [netlify.com](https://netlify.com) → **Sign up** (через GitHub или email)
2. На главной странице найди блок **"Deploy manually"** или перейди в **Sites**
3. Просто **перетащи папку с `index.html`** в браузер
4. Netlify сразу даст ссылку вида `https://random-name.netlify.app`
5. В настройках можно сменить на красивый адрес вида `polina-coffee.netlify.app`

---

## Итого — порядок действий

```
1. Создай бота → получи TG_TOKEN
2. Узнай свой chat_id → TG_CHAT
3. Вставь оба значения в index.html
4. Задеплой через GitHub Pages или Netlify
5. Отправь Полине ссылку 💌
```

## Что происходит когда Полина заполняет форму

- В твой Telegram придёт сообщение вида:
  ```
  ☕ Новая запись на кофе с Полиной!

  📅 Дата: пятница, 20 июня 2025 г.
  🕐 Время: 16:00
  📍 Место: Кофейня "Буквоед"

  Записано: 11.06.2026, 14:32:05
  ```
- На сайте появится карточка с датой, временем и надписью "Отлично, жду тебя! 🎉"
